# marmap2: Import, Plot, and Analyze Bathymetric and Topographic Data

marmap2 is an experimental modernization of marmap, a package designed
for downloading, converting, and plotting bathymetric and topographic
data in R.

## Details

The current development version focuses on a compact modern core:
NOAA/GEBCO import, tibble-based workflows, conversion helpers,
`sf`/`terra` interoperability, and `ggplot2`-based bathymetric maps.

## Reference

Pante E, Simon-Bouhet B (2013). marmap: A Package for Importing,
Plotting and Analyzing Bathymetric and Topographic Data in R. *PLoS
